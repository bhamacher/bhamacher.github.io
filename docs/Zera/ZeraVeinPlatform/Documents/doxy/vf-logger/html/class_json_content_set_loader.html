Detailed Description

The JsonContentSetLoader class.

Reads json files of the following format.

{
"ContentSet": {
"%{contentSetName}": [
{
"EntityId": "${id}",
"Components": [
${componentName},
...
]
},
},
"Sessions": {
"${sessionName}": [
${contentSetName},
...
]
}
}

To files are needed. One for defualt configuration. This file is not editable (zeraContentSetPath). And one editable file (customerContentSetPath). The ouptut of this class merges the results of both files.

zeraContentSetPath will always be priortised in case a a contentSet is available in both files.
